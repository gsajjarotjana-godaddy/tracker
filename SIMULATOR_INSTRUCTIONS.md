# View on Phone Simulator

## Method 1: Safari Responsive Design Mode (Easiest - No Xcode needed)

1. Open Safari on your Mac
2. Go to: `http://localhost:8000`
3. Press `Cmd + Option + I` (or go to Develop → Enter Responsive Design Mode)
4. Choose a device from the dropdown (iPhone 14 Pro, iPhone SE, etc.)
5. You'll see how it looks on that device!

**Shortcut:** `Cmd + Option + R` toggles responsive design mode

## Method 2: iOS Simulator (If you have Xcode)

1. Open Xcode
2. Go to Xcode → Open Developer Tool → Simulator
3. In Simulator, open Safari
4. Go to: `http://192.168.1.67:8000` (your Mac's IP)
   OR if Simulator can access localhost: `http://localhost:8000`

## Method 3: Chrome DevTools (Alternative)

1. Open Chrome
2. Go to: `http://localhost:8000`
3. Press `Cmd + Option + I` (or F12)
4. Click the device icon (toggle device toolbar)
5. Choose an iPhone model from the dropdown

## Current Server URL:
- Local: `http://localhost:8000`
- Network: `http://192.168.1.67:8000`
